# MergeDropX

A hyper-casual merge-and-drop number game built with HTML5 Canvas. Drop numbered balls, merge matching numbers to score big, and chase the highest combo!

## Play

Open `index.html` in any browser, or install as a PWA on your phone.

## Build Native (Android / iOS)

```bash
npm install
npx cap sync
npx cap open android   # Opens Android Studio
npx cap open ios       # Opens Xcode (macOS only)
```

## Tech Stack

- Vanilla HTML5 Canvas + JavaScript (zero dependencies for gameplay)
- Progressive Web App (installable on any device)
- Capacitor for native Android / iOS shells
